# Agenda de Citas para Salón de Manicura

Esta es una aplicación web sencilla para llevar una agenda de citas. Está pensada para ser desplegada en servicios de hosting gratuito como **Vercel**, **Render** o **Railway**.

## Requisitos

- Node.js 14 o superior
- npm

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```

Abrir `http://localhost:3000` en el navegador. Iniciar sesión con:

- **Usuario:** `admin`
- **Contraseña:** `1234`

Desde el calendario se pueden agregar, modificar y eliminar citas. Todo el contenido está en español.
